# AGRINHO26
# 🌱 Agro Forte, Futuro Sustentável

Projeto desenvolvido para o Concurso Agrinho 2026.

## Tema Principal

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Tema Escolhido

O que fazer para fortalecer o agro forte.

## Objetivo

Demonstrar como a tecnologia, a preservação ambiental e o uso responsável dos recursos naturais podem fortalecer o agronegócio de forma sustentável.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Apresentação do tema Agrinho 2026
- Explicação dos desafios do agronegócio
- Propostas para fortalecimento do agro
- Relação entre produção e sustentabilidade
- Quiz interativo educativo
